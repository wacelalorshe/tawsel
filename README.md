# متجر إلكتروني مجاني

متجر إلكتروني كامل مع لوحة تحكم، مبرمج بلغات HTML, CSS, JavaScript.

## المميزات
- ✅ عرض المنتجات
- ✅ لوحة تحكم لإدارة المنتجات
- ✅ تصميم متجاوب
- ✅ تخزين محلي للبيانات

## التشغيل
1. حمل الملفات
2. افتح `index.html` في المتصفح
3. للوحة التحكم: `admin/dashboard.html`

## التقنيات المستخدمة
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
